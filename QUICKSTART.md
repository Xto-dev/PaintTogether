# Paint Together - Quick Start Guide

## 🚀 Быстрый старт

### 1. Запуск приложения

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

### 2. Для разработки (с автоперезагрузкой)

```bash
npm run dev
```

## 📝 Настройка автоматической документации

### Шаг 1: Получить Claude API ключ

1. Зарегистрируйтесь на https://console.anthropic.com/
2. Создайте API ключ
3. Скопируйте ключ

### Шаг 2: Настроить локальное окружение

```bash
# Скопировать пример файла окружения
cp .env.example .env

# Отредактировать .env и добавить ваш API ключ
# CLAUDE_API_KEY=your_api_key_here
```

### Шаг 3: Тестирование локально

```bash
# Проверить работу документации на последнем коммите
python src/main.py --commit HEAD

# Сгенерировать только README
python src/main.py --only-readme

# Сгенерировать только CHANGELOG
python src/main.py --only-changelog
```

### Шаг 4: Настройка GitHub Actions

1. Перейдите в Settings → Secrets and variables → Actions
2. Добавьте секрет `CLAUDE_API_KEY` с вашим API ключом
3. GitHub Actions автоматически запустится при push в main/develop

## 🎨 Использование приложения

1. Откройте http://localhost:3000
2. Введите ваше имя
3. Введите название комнаты (создайте новую или присоединитесь к существующей)
4. Введите пароль комнаты
5. Начните рисовать!

## 🔧 Инструменты рисования

- **Цвет**: Выберите цвет из палитры
- **Размер кисти**: Настройте размер от 1 до 50 пикселей
- **Ластик**: Переключитесь в режим стирания
- **Очистить холст**: Удалить все рисунки (для всех пользователей)

## 📱 Мобильная поддержка

Приложение полностью поддерживает сенсорное управление на мобильных устройствах.

## 🔒 Система комнат

- Комнаты создаются автоматически при первом подключении
- Каждая комната защищена паролем
- Комнаты удаляются автоматически, когда последний пользователь выходит
- Все рисунки синхронизируются в реальном времени

## 🤖 Автоматическая документация

После настройки GitHub Actions:
- README.md обновляется автоматически при изменениях кода
- CHANGELOG.md генерируется на основе коммитов
- Документация коммитится обратно в репозиторий

## 🐛 Решение проблем

### Порт уже занят
```bash
# Изменить порт
PORT=8080 npm start
```

### Python скрипт не работает
```bash
# Проверить установку зависимостей
pip install -r requirements.txt

# Проверить наличие .env файла
cat .env
```

### GitHub Action не запускается
- Проверьте, что секрет `CLAUDE_API_KEY` добавлен
- Проверьте, что push идет в ветку main или develop
- Проверьте логи в разделе Actions
